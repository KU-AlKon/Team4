## ✅ 소스코드 파일 이름 규칙
- `{백준 문제 번호}-{티어(영문)}` 형식으로 작성합니다.
- 예를 들어, `2577-Bronze2` 처럼 파일 이름을 정하시면 됩니다.
- 티어 영문 표현
  - Bronze
  - Silver
  - Gold
  - Platinum
  - Diamond
  - Ruby
