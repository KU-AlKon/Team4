## ✅ 소스코드 파일 이름 규칙
- `{백준 문제 번호}-{티어(영문)}` 형식으로 작성합니다.
- 예를 들어, `2577-Bronze2` 처럼 파일 이름을 정하시면 됩니다.
- 티어 영문 표현
  - Bronze
  - Silver
  - Gold
  - Platinum
  - Diamond
  - Ruby

## ✅ commit 규칙(Optional)
- commit 메세지: `[문제 번호] 문제이름 / 티어`
- 터미널에서 작성법:

## ✅ PR 규칙
- PR 제목: 이름 / 날짜 / 문제 번호
-  예시) `홍승택 / 3월 19일 / 1451 `
-  내용: PR 템플릿에 맞추어 작성합니다.
    - 문제 번호
    - 백준 URL
    - 문제 티어 
    - COMMENT: 생각하는 과정, 어려웠던 점, 배우된 점 등을 자유롭게 서술합니다.
